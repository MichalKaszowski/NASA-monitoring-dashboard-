# NASA-monitoring-dashboard
Nokia recruitment project - monitoring dashboard of asteroids passing near Earth

Link to deployed page [(GitHub Pages)](https://MichalKaszowski.github.io/nasa-monitoring-dashboard)

## 1. Features
The app consists of following features: 
- fetching asteroids data by date range from (NASA's API)[https://api.nasa.gov/] and display them on a table, with sorting and filtering functionality.
- Data visualization with risk highlight through charts:
a) Top 5 biggest asteroids (by estimated max diameter) with comparison to Earth's buildings

b) Hazard map - speed and lunar distance display with information if the asteroid is hazardous.
c) Proportion of hazardous and non-hazardous asteroids
